# 日志管理

---

## 使用说明

### 功能概述
> 本服务为其他服务模块提供日志管理功能。可以完成如下工作：
> - 新增日志（单个）
> - 查询日志（单个／列表）
> - 删除日志（单个）

### 请求地址
    [HTTP动词]:http:ip:port/application/api/[版本号]/[URI]

> - 所使用的`HTTP动词`有如下三种：
    （1）`POST`：增加
    （2）`GET`：查询
    （3） `DELETE`：删除
> - 目前维护的版本号为`v1`
> - 各`URI`详见具体功能说明
